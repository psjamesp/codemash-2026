## üöÄ Quick Start for Instructors

1. **Review MASTER-INSTRUCTOR-GUIDE.ps1** - This is your teaching bible
2. **Test all demo scripts** on your machine first
3. **Share Demo*.ps1 files with students** before or at the start of class
4. **Keep energy high** - this is hands-on, not lecture

---

## üí° Key Teaching Tips

- **Walk around during practice time** - students need help but won't always ask
- **Celebrate errors** - "Great! Let's debug this together"
- **Keep breaks on schedule** - people need mental breaks
- **Share your own stories** - why YOU learned PowerShell
- **Don't skip practice sections** - hands-on time is where learning happens
